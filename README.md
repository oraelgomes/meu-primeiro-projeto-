# meu primeiro projeto 

## Como utilizar o projeto

Este projeto foi desenvolvido para demonstrar conceitos básicos de entrada de dados em Python, leitura de informações digitadas pelo usuário e realização de um cálculo simples. A aplicação funciona diretamente no terminal, solicitando algumas informações ao usuário e exibindo os resultados na tela.

Ao executar o programa, o sistema começará pedindo que o usuário digite seu nome. Depois que o nome é informado, o programa o exibe na tela para confirmar que o dado foi recebido corretamente.

Em seguida, o programa solicita que o usuário digite uma nota. Após o usuário inserir o valor, o sistema imprime uma mensagem mostrando o nome informado anteriormente junto com a nota digitada.

Na terceira parte da execução, o programa pede que o usuário informe o valor de um produto. Esse valor é inicialmente recebido como texto, pois todo dado obtido pela função input() em Python é tratado como string. Para que seja possível realizar um cálculo matemático, o valor é convertido para número inteiro utilizando a função int().

Após a conversão, o programa calcula um desconto fixo de 5 unidades sobre o valor informado do produto. O resultado desse cálculo é então exibido no terminal para o usuário.

Para utilizar o projeto, basta executar o arquivo Python no terminal e seguir as instruções exibidas na tela, digitando as informações solicitadas. O programa processará os dados e mostrará os resultados imediatamente.

Este projeto é indicado para quem está começando a aprender Python, pois demonstra de forma simples como trabalhar com:

- entrada de dados do usuário
- exibição de informações no terminal
- conversão de tipos de dados
- operações matemáticas básicas
